🔧 Airco Beheer - Onderhoudsrapport Systeem

📋 Project Overzicht: Volledig werkend onderhoudsrapport systeem voor Airco Beheer met admin panel en klantrapport pagina.

📌 Belangrijke URLs

  • Admin Panel: http://localhost/site/admin.php
  • Rapport Pagina: http://localhost/site/rapport.php?id=UUID
  • phpMyAdmin: http://localhost/phpmyadmin

1️⃣ Admin Panel (admin.php)

Functionaliteit:

  • Formulier voor het aanmaken van nieuwe onderhoudsrapporten
  • Velden:
    • Klantnaam (verplicht)
    • Datum bezoek (verplicht)
    • Locatie/Adres (verplicht)
    • Status: A (Uitstekend), B (Goed), C (Aandacht vereist)
    • Uitgevoerde werkzaamheden (bullet points)
    • Opmerking monteur (optioneel)
    • Foto uploads (max 12 foto's)
  • Na opslaan: genereert unieke link per klant
  • "Kopieer link" knop om naar klant te sturen via e-mail
  • "Nieuw rapport aanmaken" knop om formulier te resetten

2️⃣ Klant Rapport Pagina (rapport.php)

Functionaliteit:

  • Laadt rapportgegevens dynamisch uit database op basis van unieke ID
  • Toont:
    • Status badge (A/B/C met kleurcodering)
    • Samen vatting van uitgevoerde werkzaamheden
    • Foto's met lightbox viewer (klik om te vergroten)
    • Opmerking monteur
    • "Activeer jaarlijks onderhoud" CTA knop
  • Als ID niet bestaat: toont "Rapport niet gevonden" bericht

3️⃣ Foto Uploads

  • Fotos worden geüpload via admin formulier
  • Opgeslagen op server in uploads/fotos/
  • Elke foto heeft eigen label (bijv. "Binnenunit voor", "Binnenunit na")
  • Maximaal 12 foto's per rapport
  • Ondersteunde formaten: JPG, PNG, WEBP

4️⃣ Database (MySQL)

Database naam: airco_beheer

Tabellen:

rapporten (onderhoudsrapporten)

Veld Type Beschrijving
id VARCHAR(36) UUID unieke identificator
klant VARCHAR(255) Klantnaam
locatie VARCHAR(500) Adres/locatie
datum DATE Datum van bezoek
status ENUM('A','B','C') Status van installatie
samenvatting JSON Lijst van uitgevoerde werkzaamheden
monteur_opmerking TEXT Opmerking van monteur
fotos JSON Array van foto objecten met url en label
created_at DATETIME Aanmaakdatum

contract_aanvragen (upgrade verzoeken)

Veld Type Beschrijving
id INT Auto increment ID
rapport_id VARCHAR(36) Verwijzing naar rapport
aangevraagd_op DATETIME Wanneer aangevraagd

⚙️ Installatie

Stap 1: Database Installeren

  1. Open phpMyAdmin: http://localhost/phpmyadmin
  2. Maak een nieuwe database aan: airco_beheer
  3. Klik op "Import" en selecteer het bestand database_setup.sql
  4. Klik op "Go" om de tabellen aan te maken

Stap 2: Bestanden Controleren

Zorg dat de volgende bestanden in de root staan:

admin.php - Admin panel rapport.php - Klant rapport pagina api.php - Backend API config.php - Configuratie database_setup.sql - Database schema uploads/fotos/ - Map voor foto uploads

📖 Gebruik

Stap 1: Maak Rapport aan

  1. Ga naar admin.php
  2. Vul klantgegevens in
  3. Selecteer status
  4. Voeg werkzaamheden toe (klik "+ Regel toevoegen")
  5. Upload foto's
  6. Klik "Rapport opslaan & link genereren"

Stap 2: Deel Link met Klant

  1. Na opslaan verschijnt een succes scherm
  2. Klik "Kopieer link"
  3. Plak in e-mail naar klant

Stap 3: Klant Bekijkt Rapport

  1. Klant opent de link
  2. Ziet alle rapportgegevens
  3. Kan foto's bekijken in lightbox
  4. Kan "Activeer jaarlijks onderhoud" klikken

🔧 Technische Details

Base URL

De base URL is centraal gedefinieerd in config.php:

define('BASE_URL', 'http://localhost/site');

API Endpoints

Actie Methode URL
Get Rapport GET api.php?action=get_rapport&id=UUID
Create Rapport POST api.php (action=create_rapport)
Upload Foto POST api.php (action=upload_foto)
Activate Contract POST api.php (action=activate_contract)
